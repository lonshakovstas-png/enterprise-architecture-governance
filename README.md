# Enterprise Architecture Governance

This repository is the authoritative baseline for the Enterprise Architecture Governance Framework. All governance artifacts are maintained here and constitute the single source of truth for architecture governance within the Digital Solutions function.

## Framework Status

| Field   | Value                                    |
| ------- | ---------------------------------------- |
| Version | 1.0                                      |
| Status  | Approved                                 |
| Owner   | Digital Solutions Function               |
| Authority | ADR-0011 — Architecture Governance Framework v1.0 |

## Repository Structure

| Directory       | Contents                                             |
| --------------- | ---------------------------------------------------- |
| `01_ADR/`       | Architecture Decision Records                        |
| `02_Policies/`  | Governance policies and charters (DOC)               |
| `03_Regulations/` | Governance procedures and processes (REG)          |
| `04_Standards/` | Normative architecture standards (STD)               |
| `05_Templates/` | Approved document templates (TPL)                    |
| `06_Registers/` | Framework registers (RGS)                            |
| `07_Glossary/`  | Terms and definitions (GLO)                          |
| `99_Archive/`   | Withdrawn and superseded artifacts                   |

## Document Authority Hierarchy

ADR > DOC > REG > STD

In cases of conflict, the higher-level document prevails.
